# Create Yuki App

A Next.js template with TypeScript, Tailwind CSS, ESLint and Prettier.

## Getting Started

```bash
npx create-next-app@latest --example https://github.com/tiesen243/create-yuki-app
```

## Features

- TypeScript
- ESLint
- Prettier
- Tailwind CSS
